# DanialBazaar
Bag-Shoe store using payment through Zarinpal website

## Tech stacks
Jetpack Compose, MVVM pattern, Kotlin Coroutines, Navigation compose, Livedata, Coil, Koin, Retrofit2, Room, Accompanist, Lottie
